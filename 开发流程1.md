# 三 登录

## （一）前端

```tex
【提示词】
【请基于当前已经存在的 Spring Boot 3.x + MyBatis 项目继续开发，不要删除或修改现有后端功能，在现有项目基础上新增前端页面框架，并同步更新 README.md。
项目名称：威胁感知平台；英文名称：Threat Awareness Platform。
一、总体要求
1. 使用 Layui 实现页面框架。
2. 页面全部使用中文。
3. 风格参考 Claude、Linear、Notion 等现代 SaaS 产品。
4. 整体采用现代极简设计语言。
二、前端目录结构
CSS、JS、HTML分离。
三、登录页面
1. 登录表单仅保留：用户名、密码、登录按钮。表单字段与用户表对应
2. 表单校验：用户名不能为空、密码不能为空。
3. 登录按钮点击后调用：POST /api/user/login。
4. 登录成功：保存 token 至 localStorage，跳转至：index.html。登录失败，使用 Layui message提示错误信息。
四、后台主框架
布局采用经典后台管理系统布局：Header、Sidebar、Main Content。
五、顶部导航栏 Header
左侧显示：威胁感知平台；右侧显示：当前登录用户；退出登录按钮。
点击退出：清除 localStorage 中 token，跳转登录页。
六、左侧菜单 Sidebar
菜单使用 Layui 导航组件。菜单如下：首页、用户管理、威胁分析、风险评估、检测记录、系统日志、系统设置。
要求：支持菜单折叠。支持菜单高亮。支持 iframe 或内容区域切换。后续新增菜单时无需修改整体框架。】
```

仍是静态页面，预留接口，都是假页面。
